# 🛒 E-Commerce Sales Analysis using MySQL

## 📌 Project Overview

This project analyzes an E-Commerce dataset using MySQL to answer real-world business questions. The analysis focuses on customer behavior, order trends, and product performance.

---

## 🎯 Objectives

- Analyze customer purchase behavior
- Identify top-selling products
- Find top customers
- Analyze daily sales trends
- Practice SQL for business analysis

---

## 📂 Dataset

The project contains two datasets:

- customer_details.csv
- basket_details.csv

---
